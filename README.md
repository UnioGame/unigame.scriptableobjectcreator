# ScriptableObject Creator

## Overview

![Window](https://i.ibb.co/3y2RnMGF/socreator2.png)

A lightweight UI Toolkit-based editor window for creating ScriptableObject instances

## Features

- **Type Discovery**: Automatically finds all ScriptableObject types in the project
- **Search Functionality**: Filter types by namespace and class name
- **Asset Creation**: One-click asset generation with automatic unique naming

## Installation


```json
"com.unigame.scriptableobjectcreator" : "https://github.com/UnioGame/unigame.scriptableobjectcreator.git",
```

## Usage

### Opening the Window

1. Right-click in the Project window
2. Select **Create New Scriptable Object**
